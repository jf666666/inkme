
# inkme

## Introduction

inkme is a SplatNet 3 client developed in Swift, currently in active development. This project aims to provide the core features of SplatNet 3 and plans to introduce innovative features such as historical data storage, multiple account management, etc., to provide Splatoon players with a more comprehensive and convenient experience.

## Current Status

inkme is under development, working towards implementing the basic features of SplatNet 3. Please note that some features may still be incomplete or not yet implemented.

## Planned Features


| Feature                | Status    | Notes                            |
|------------------------|-----------|----------------------------------|
| Schedule Query         | Mostly Complete | Interface for competitive matches not yet designed |
| Historical Play Record Storage | Mostly Complete | Implemented storage for work records and battle records |
| Historical Play Record Statistics | Not Implemented | - |
| Multiple Account Management | Mostly Complete | Multi-account management functionality still has bugs to be resolved |
| Sharing Functionality | Partially Complete | Currently, only the work interface save function is implemented |

## Getting Started

To start using inkme, you can follow these steps to download and set up the project:

### Downloading the Project

1. **Download ZIP**:
   - Visit the project's GitHub page.
   - Click on the “Code” button, then select “Download ZIP”.
   - Unzip the file to your desired folder.

2. **Using Xcode**:
   - Open Xcode.
   - Select “Clone an existing project”.
   - Enter the GitHub repository URL of the project and follow the prompts.
   - Or click [Open in Xcode](xcode://clone?repo=https%253A%252F%252Fgithub.com%252Fjf666666%252Fimink3)

### Setting Up the Project

Once you have downloaded the project, you can open the `.xcodeproj` file in the project folder using Xcode. If the project has any dependencies, you may need to install and configure them according to the project documentation.

### Running the Project

After opening the project in Xcode, you can select an appropriate simulator or connect your device, then click the run button to start the application.
